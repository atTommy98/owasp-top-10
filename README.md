## What is OWASP?
The Open Worldwide Application Security Project (OWASP) is an open community dedicated to helping organisations and individuals develop, purchase and maintain applications and APIs that can be trusted. The OWASP project provides a plethora of tools and information such as:
- Application security tools and standards to adhere to.
- Cutting edge security research
- Software security protocols and libraries
- Books, presentations, videos and cheat sheets on many common topics and security testing, code development and secure code reviews
